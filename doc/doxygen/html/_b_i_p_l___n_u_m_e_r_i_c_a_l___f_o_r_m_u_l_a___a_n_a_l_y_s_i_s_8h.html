数値，数式解析

パラメータ設定に独自形式を使うかどうかが分からないが，念のため．

BNF記法 AdditionAndSubtraction :: = <MultiplicationAndDivision> | {<MultiplicationAndDivision>{+|-}{<AdditionAndSubtraction}>} MultiplicationAndDivision :: = <Pulus_MinusSign> | {<Pulus_MinusSign> {*|/} <MultiplicationAndDivision>} Pulus_MinusSign :: = {+|-| } <Power> Power :: = <factor> | {<Power> {^} <Factor>} Factor :: = <Number> | (<AdditionAndSubtraction>) | 特殊定数(頭文字N_)

使用例 std::string formula = "10+(1+2+3*4+5)/(4^2)"; std::cout << formula << " = " << BIPL_NumericalFormulaAnalysis(formula) << std::endl;
