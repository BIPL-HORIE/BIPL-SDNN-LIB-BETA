詳解

template<class kernel_type, class output_type>
class BIPL_RANDOM< kernel_type, output_type >

一様分布の乱数．実数一様か整数一様かは，テンプレートで指示．

Initした後は，普通の変数と同じように使える． 本クラスは直接実体化することをお勧めしない． BIPL_RANDOM_UNIFORM_INTか，BIPL_RANDOM_UNIFORM_DOUBLEで実体化することを勧める．
