詳解

SDNNのNN部分，これは継承前提の仮想クラス

出力される2値パターンは，1素子＝1bitで表し32素子ごとに管理． 50素子からなるパターンは， 一つ目のunsinged int に，上位32素子 二つ目のunsigned int に，下位18素子が下詰めで表現．

SDでは，配列を昇順に，下位ビットから処理
